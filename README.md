# vue-plugin-vuetify-base

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles as a library and minifies for production
```
npm run build
```
#### Compiles as a library, minifies and create a tarball file
```
npm run build:pack
```
#### Build the demo page
```
npm run build:demo
```

### Lints and fixes files
```
npm run lint
```
